# wishlist

Wishlist is a web application to create your wishlists with wishes.
You can share wishlists with other users by giving them wishlist ID and password 


Technologies: 
- Java 17
- Spring Boot
- Spring Web
- Spring Security
- Spring Data JPA
- Thymeleaf
- Validation
- Lombok
- PostgreSQL
- H2
- JUnit 5
- Mockito
